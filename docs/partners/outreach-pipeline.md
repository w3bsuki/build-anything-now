# Outreach Pipeline (Clinics, NGOs, Stores)

- Owner: Partnerships
- Update cadence: Weekly
- Last updated: 2026-02-06

## Goal
Build a trusted Bulgaria partner network that supports rescue-to-adoption completion.

## Target Segments
1. Vet clinics (highest priority)
2. NGOs/rescue organizations
3. Pet stores and supply partners
4. Shelters and foster networks

## Pipeline Stages
- Prospect identified
- Contacted
- Intro call scheduled
- Pilot accepted
- Profile claimed/verified
- Active partner

## Standard Outreach Pack
- 1-page value proposition
- Claim profile instructions
- Verification and trust FAQ
- Launch-city playbook and expected response SLA

## Messaging Framework
- For clinics: faster funded intake + visibility + trust badges.
- For NGOs: transparent case funding + update timeline.
- For stores: sponsorship and supply-impact visibility.

## Weekly Cadence
- Mon: new prospect upload
- Tue-Thu: outreach calls/emails
- Fri: conversion review + blockers

## Metrics
- Contact-to-call conversion
- Call-to-pilot conversion
- Claim completion rate
- Active partner retention
