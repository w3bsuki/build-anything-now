# Product Roadmap (8-Week Delivery)

- Owner: Product (Human) + OPUS
- Update cadence: Weekly
- Last updated: 2026-02-06

## Sequencing Rule
Priority order is fixed:
1. Trust + money + case lifecycle
2. Directory expansion + partner operations
3. Mission initiatives surfaces

## Phase 0 (Week 1) — Baseline and Governance Reset
### Deliverables
- Canonical docs cleanup and supporting docs pack under `docs/`.
- Decision log updates for locked assumptions.
- Baseline quality check and debt list capture.

### Exit Criteria
- Canonical links resolve.
- No references to archived `.specs` in active workflow docs.
- Supporting plan docs created with ownership and cadence.

## Phase 1 (Weeks 2-3) — Case Lifecycle Completion
### Deliverables
- Owner/authorized update composer on case detail.
- Structured updates with evidence metadata.
- Lifecycle transitions including `seeking_adoption` and closure outcomes.

### Exit Criteria
- Owner can post updates with optional media evidence.
- Supporters see timeline updates immediately.
- Lifecycle transitions enforced and persisted.

## Phase 2 (Weeks 3-5) — Trust + Money
### Deliverables
- Stripe payment intent + webhook completion path.
- Idempotent donation completion and receipt metadata.
- Moderation queue actions and verification updates with audit logs.

### Exit Criteria
- End-to-end small donation in Stripe test mode.
- Verification state updates visible in feed and detail.
- Admin/mod ops can process reports safely.

## Phase 3 (Weeks 5-7) — Bulgaria Directory + Partner Ops
### Deliverables
- Seeded clinic dataset for Sofia, Varna, Plovdiv.
- Active claim flow and claim review operations.
- Outreach pipeline for NGOs/clinics/stores.

### Exit Criteria
- Clinic discovery works by city and 24/7 filters.
- Claim duplicates blocked; review lifecycle traceable.
- Outreach process is documented and actionable.

## Phase 4 (Weeks 7-8) — Mission Campaign Surfaces
### Deliverables
- Home featured initiative module.
- Account/profile mission hub.
- Clear separation of rescue campaigns vs platform initiatives.

### Exit Criteria
- Users can discover/donate to initiatives distinctly.
- Initiative roadmap and use-of-funds are explicit.

## Monitoring and Rollout
- Cohort launch focus: Sofia, Varna, Plovdiv.
- Weekly metric review: case creation speed, trust conversion, moderation SLA, claim approval time.
- Feature flags recommended for donations, moderation, lifecycle transitions.
